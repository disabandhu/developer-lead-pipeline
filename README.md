# DEV.to Developer Intelligence Pipeline

A Python-based developer discovery and enrichment pipeline built using the DEV.to API.

## Features

- Developer discovery from DEV.to articles
- Profile collection via DEV API
- Website classification
- Website enrichment
- Email extraction
- LinkedIn extraction
- GitHub extraction
- Lead scoring

## Results

Current dataset:

- 632 developer profiles collected
- 406 websites discovered
- 152 emails extracted
- 195 LinkedIn profiles found
- 209 GitHub profiles found

## Tech Stack

- Python
- Requests
- BeautifulSoup4
- JSON

## Pipeline

DEV Articles API
→ Username Discovery
→ Profile Collection
→ Website Enrichment
→ Contact Discovery
→ Lead Scoring
